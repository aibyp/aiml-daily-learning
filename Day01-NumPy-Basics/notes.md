# Day 01 – NumPy Basics

## What is NumPy?
NumPy (Numerical Python) is a library used for fast numerical computations.  
It provides support for arrays, matrices, and mathematical functions.

## Why NumPy is used in AI/ML?
- Faster than Python lists
- Supports multi-dimensional arrays
- Used in TensorFlow, PyTorch, Pandas
